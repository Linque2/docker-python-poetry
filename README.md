A robust, configurable `python-poetry` Docker image.


## Build
You can build this `python-poetry` image yourself using any version of Poetry
and Python.

```bash
make build POETRY_VERSION="1.1.13" PYTHON_IMAGE_TAG="3.8.13-slim"
```
